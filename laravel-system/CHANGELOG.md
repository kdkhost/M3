# Changelog

Todas as alterações significativas no projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Semantic Versioning](https://semver.org/lang/pt-BR/).

## [1.0.0] - 2026-06-26

### Added

- Criação inicial do projeto com Laravel 13
- Instalação e configuração do AdminLTE 4
- Instalação do Laravel Sanctum para autenticação de API
- Instalação do Spatie Laravel Permission para RBAC
- Instalação do Spatie Laravel Backup para backups automáticos
- Instalação do Spatie Laravel Media Library para gerenciamento de mídia
- Instalação do Yajra DataTables para listagens com Server Side
- Instalação do Intervention Image para manipulação de imagens
- Configuração do timezone para America/Sao_Paulo
- Configuração do locale para pt_BR
- Criação do README.md com instruções de instalação e deploy
- Criação do CHANGELOG.md
